# 👟 Imperial Mogi Modas - E-commerce Landing Page

Uma landing page premium desenvolvida para a **Imperial Mogi Modas**, focada na apresentação de calçados multimarcas de alta qualidade. O projeto combina um design sofisticado com uma interface intuitiva para proporcionar a melhor experiência de compra ao usuário.
---
## 🚀 Tecnologias Utilizadas
O projeto foi construído utilizando as melhores práticas de desenvolvimento front-end:
* **HTML5**: Estruturação semântica de dados.
* **CSS3**: Estilização avançada com foco em Flexbox e Grid.
* **JavaScript**: Manipulação de DOM para interatividade.
* **Responsividade**: Design adaptável para dispositivos móveis e desktops.
* **Google Fonts**: Tipografia moderna (Montserrat/Poppins).
---
## 🎨 Diferenciais do Projeto
### 💎 Design Premium
Utilização de uma paleta de cores sóbria (Preto, Branco e Dourado) para transmitir exclusividade e sofisticação, alinhada ao público-alvo da marca.
### 📱 Foco em Conversão (UX/UI)
* **CTA Estratégico**: Botões de "Tenho Interesse" integrados diretamente para facilitar o contato via WhatsApp.
* **Seção de Destaques**: Grid organizado com cards detalhados exibindo marca, modelo e numeração disponível.
* **Navegação Fluida**: Menu fixo e intuitivo para acesso rápido às seções da loja.
